# StayChat AI Task

## Overview
This project is a full-stack CRUD application built using Flask, MongoDB Atlas, and React.

## Tech Stack
- Backend: Python Flask
- Database: MongoDB Atlas
- Frontend: React
- API Communication: Fetch API

## Features Implemented
- Create or Update Item (Upsert)
- Get All Items
- Delete Item by ID
- Timestamp stored in database

##  How to Run Locally

### Backend
cd backend
pip install flask flask-pymongo flask-cors pymongo[srv]
python app.py

Runs on: http://127.0.0.1:5000

### Frontend
cd frontend
npm install
npm start

Runs on: http://localhost:3000
